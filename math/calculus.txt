Числовая последовательность -------------------------------------------------
Определение:
    Если каждому натуральному числу n∈N по некоторому правилу xn = f(n)
    поставлено в соответствие опред действит число xn∈R, то говорят
    задана числовая последовательность x1,x2,x3,...,xn,...
    x1,x2,x3,...,xn,... # члены последовательности
    xn                  # общий член последовательности
    {xn}, n∈N           # задается послед-ть {форм общ члена}, n приним нат знач

Example:
    x: 2,4,6,8,10...    # дана последовательность x
    xn = 2n             # xn=2n - правило(формула), n∈N(множ натур чисел)
    Каждому натуральному числу n=1,2,3,... по правилу (xn = 2n) поставлены в
    соответствие числа 2,4,6,8,10... => это числовая послед-ть
    x10 = 2*10 = 20     # вычисление значения 10-го члена последовательности

Ограниченная последовательность:
Определение:
    Последовательность {xn} наз-ся ограниченной, если сущ такое число M>0, что
    для любого номера n∈N будет выполняться неравенство |xn|<=M.
    В противном случае последовательность - неограниченная.
Кванторы:
    ∃M>0: ∀n∈N |xn|<=M

Example:
    xn = 1/n    # возьмем M=1 ,|xn|<=M - след-но это огранич посл

Монотонные последовательности:
Возрастающая последовательность:
    Если для последовательности {xn} при любом n∈N выполняется неравенство
    x[n+1]> xn, то посл-ть наз-ся возрастающей, кажд след член больше предыдущ.
Неубывающая последвательность:
    x[n+1]>= xn
Убывающая последовательность:
    x[n+1]< xn
Невозрастающая последовательность:
    x[n+1]>= xn

Немонотонная последовательность:
    не подходит к правилам монотонной послед-ти
    xn = (-1)**n * n    # {-1,2,-3,4,-5,...}

Способы задания последовательностей:
Аналитический. Задаётся формулой общего члена xn:
    Example: xn = 2n
    x1 = 2*1 = 2 
    x2 = 2*2 = 4
    x3 = 2*3 = 6
    ...
    x[n-1] = 2*(n-1) = 2n-2
    xn = 2n
    x[n+1] = 2*(n+1) = 2n+2
Реккурентной формулой. Задаётся а1 или несколько первых членов, задаётся
формула по которой находятся следующие члены через предыдущие.
    Example: a[n+1] = a[n] + d
Словесным описанием

an = {1,2,3,4,5}        # конечная ограниченная, возрастающая последовательность
bn = {2,1,0,-1,-2,...}  # бесконеч убывающ, неограниченная последовательность
cn = {..-3,-5,6,8,-5..} # ни возраст ни убыв, неогранич посл
dn = {7,7,7,7,...}      # стационарная, огранич посл

Арифметическая прогрессия - числовая последовательность, в которой каждый последующий член равен предыдущий+d
    a[n+1]  = a[n] + d
    d - разность арифметической прогрессии(заданное число)
    d > 0 - возрастающая прогрессия
    d < 0 - убывающая    
    an = a1 + d*(n-1)       # Формула общего n-го члена
    Sn = (a1+an)*n/2        # Формула суммы членов прогрессии

Геометрическая прогрессия - числовая последовательность, в которой каждый
последующий член равен предыдущий*q
    b[n+1] = b[n] * q
    q - знаменатель геометрич прогрессии(заданное число)
    bn = b1 * q**(n-1)      # Формула общего n-го члена
    Sn = b1*(q**n -1)/(q-1) # Формула суммы членов прогрессии
    Sn = (bn*q-b1)/(q-1)    # 2 Формула суммы членов прогрессии
    S = b1/(1-q), q!=1      # Сумма бесконечно убывающей геом прог (|q|<1)
    Sn = n*b1               # Формула суммы членов прогрессии if q==1

Предел последовательности
          /    ε - окрестность    \         # a - предел последовательности
    -----+------------+-----+------+-----   # (a-ε;a+ε) - окрестность
        a-ε           a     x10   a+ε       # ε - выбир-ое нами знач, положит

определение:
    пределом последовательности xn наз-ся число а, если для любого числа
    положительного - эпсилон(выбранного нами, сколь угодно малого),
    существует такой номер N (номер члена последовательности)(зависящий от
    эпсилон), принадлежащий  множеству натуральных чисел N(жирное), 
    такой что начиная с этого номера - все большие номера n(малое)
    т.е. их члены последов-ти лежат в окрестности эпсилон: |xn-a| < ε
пояснения:
    |xn-a| < ε      # по правилу модуля расписываем 
    -ε < xn-a < ε   # опускаем модуль
    a-ε < xn < a+ε  # выносим а за знаки нерав-ва - получим наш интервал(окрест)
    N(ε)            # N зависит от ε - это значит что N находится через нерав-во
                      в котором фигурирует и эпсилон, N = [|xn-a|<ε], целая
                      часть от неравенства  |xn-a|<ε, в общем если берем разные
                      эпсилон, то и номер N тоже будет разным.

определение через кванторы:
    lim xn = a ⇔ ∀ε>0, ∃N(ε)∈N : ∀n>N(ε) ⇒ |xn-a|<ε
    n→∞

Сходящяяся последовательность:
    Это последовательность имеющая предел.
    Всякая сход послед имеет 1 предел.

Расходящаяся последовательность
    Последовательность не имеющая предела, т.е. lim xn = ∞ при n→∞

Example 1:
    xn = 4+1/3**n, n∈N      # последовательность, заданная формулой общ члена
    lim xn = 4?             # доказать что предел = 4
    ε = 0.00001             # установим эпсилон
    (4-0.00001;4+0.00001)   # интервал - эпсилон окрестность от точки 4
    4-0.00001i<xn<4+0.00001 # подставляем из определения
    xn>4+0.00001            # т.к. наши члены всегда >4, достаточ решить это нер
    4+1/3**n>4+0.00001      # подставим нашу послед-ть в xn
    n > log3 100000         # log по основанию 3 от 100000
    n > 10.48               # номер n долж быть больше 10.48 и целое ==> это 11
    ∀n >= 11                # для всех номеров послед xn начиная с 11 номера
                              знач xn будут попадать в эпс окрест-ть x11,x12... 

Example 2:
    xn = (2n-1)/(4n+5)      # последовательность
    найти кол-во членов последовательности лежащих вне интервала:
    (1/2-1/1000,1/2+1/1000) # интервал - эпс окрестность
    a = 1/2                 # предел посл-ти(находится, либо видно из условия)
    ε = 0.001               # эпсилон из условия
    |xn-a|<ε                # подставим все в формулу
    |(2n-1)/(4n+5)-1/2|<1/1000  # подставляем и находим n
    |-7/(8n+10)| < 1/1000   # приводим к общ знам, убираем модуль и минус
    (8n+10)/1000 > 7        # переносим, выражаем n
    8n+10 > 7000            # выражаем
    8n > 6990               # the last step)
    n > 873.75              # N=[873.75]=873(цел часть) т.к. N(больш)∈N(натур)
                              получ что после 873-го все n войдут в окрестность
    Ответ: 873 члена посл-ти лежат вне заданного условием интервала

Бесконечно малая последовательность:
    Последовательность, члены которой с увеличением номера безгранично 
    приближаются к 0.
    lim αn = 0 ⇔ ∀ε>0, ∃N(ε)∈N : ∀n>N(ε) ⇒ |αn|<ε
    n→∞
Example:
    xαn = 1/n - члены стремятся к 0 справа, оставаясь всегда >0, lim xn = 0+
    xαn = 1-/n- возрастающая, lim xn = 0-

Свойства бесконечно мылых последовательностей:
    бм ± бм = бм    # {αn±αn}
    бм * ог = бм    # {αn*yn}
    бм * C = бм     # {αn*5}
    бм * бм = бм    # {αn*αn}
    бм / бм - может быть не бм

Бесконечно большие последовательности(расходящаяся)
    Последовательность {xn} называется бесконечно большой, если для любого
    сколь угодно большого положительного M можно указать номер последоват-ти
    N, такой что все значения >N+1 по модулю будут больше M.
    |xn| → ∞ при n→∞    # значения посл-ти стремятся к беск при n стрем к беск
    lim xn =  ∞ ⇔ ∀M>0, ∃N∈N : ∀n>N ⇒ |xn|>M    # бесконеч больш посл
    lim xn = +∞ ⇔ ∀M>0, ∃N∈N : ∀n>N ⇒ xn>M      # возрастающая
    lim xn = -∞ ⇔ ∀M>0, ∃N∈N : ∀n>N ⇒ xn<-M     # убывающая

Свойства сходящихся последовательностей
    lim xn = a, xn→a, n→∞           # a - предел посл-ти {xn} при n→∞
    (xn - a) → 0, xn-a = αn         # xn-a = бм(беск мал посл)
    xn = a + αn                     # преде + бм
    1) if xn = c => lim c = c       # если посл постоянна, пред = константе
    2) if xn→a, yn→b => xn+yn→a+b   # предел суммы посл-й = сумме пределов
       lim(xn+yn) = limxn + limyn     
    3) if xn→a, yn→b => xn*yn→a*b   # предел произвед посл = произв пределов
       lim(xn*yn) = limxn * limyn   
    4) lim(c*xn) = c*limxn          # следствие из 3 и 1 св-ва
    5) if xn→a, yn→b => xn/yn → a/b # при b = limyn != 0
       lim(xn/yn) = limxn/limyn

Предельный переход в неравенствах
    1) if xn>=b: limxn>=b
       if xn<=yn: limxn<=limyn
    2) теорема о 2-х милиционерах: 
       последовательность между посл-ми стремится к тому же пределу
       limxn=limzn=a, if xn<=yn<=zn: limyn=a

Oтличия предела последовательности от предела функции
    В пределе последовательности «динамическая» переменная «эн» может
    стремиться только к «плюс бесконечности» – в сторону увеличения 
    натуральных номеров, т.к. члены последовательности пронумерованы.
    В пределе функции «икс» может быть направлен куда угодно – к «плюс/минус
    бесконечности» либо к произвольному действительному числу.

Предел функции f(x)----------------------------------------------------------
    Предел функции по Гейне: число A называется пределом функции f(x) 
    в точке x0, если для любой последовательности точек x1,x2,x3..,xn,..
    (принадлежащих X и отличных от x0), которая сходится к точке x0,
    соответствующая последовательность значений функции f(x1),f(x2)..,f(xn),...
    сходится к A.
    Предел функции по Коши: число A называется пределом функции f(x)
    в точке x0, если для любой заранее выбранной окрестности ε>0(сколь угодно 
    малой), существует δ-окрестность точки x0, ТАКАЯ, что: КАК ТОЛЬКО значения
    (принадлежащие X) входят в данную окрестность: 0<|x-x0|<δ(красные стрелки)
    – ТАК СРАЗУ соответствующие значения функции гарантированно зайдут в 
    ε -окрестность: |f(x)-A|<ε(синие стрелки).
Example:
    lim f(x)
    x→+∞
    f(x) - бесконечно приближ к своей асимптоте
Example:
    пытаемся просто подставить значение x в функцию
    lim x² = 3²
    x→3 
x может стремиться к нулю слева и справа от оси OY
    lim 1/x = +∞  # справа  
    x→0+   
    lim 1/x = -∞  # слева
    x→0-

Неопределенности типа ∞/∞
    lim 2x²-3x-5   ∞   2
    x→∞ -------- = - = -
        1+x+3x²    ∞   3
    Короткое решение:
        отбрасываем все младшие слагаемые => 2x²/3x² => 2/3
        если не сокращатся до конца - то делим на x в старш степени и 
        подставляем то к чему стремится x(в данном примере ∞).
Неопределенности типа ∞ - ∞
    Решение:
    приведением выражения под знаком предела к общему знаменателю
    умножением/делением на сопряжённое выражение
    преобразованием логарифмов
Неопределенности типа 1**∞
    устраяется по формуле
    lim u(x)**v(x) = e**lim[(u(x)-1)*v(x)]
    x→a                 x→a
    пример
    lim(1-1/x)**4x = 1**∞ = e**lim[(1-1/x-1)*4x] = e**lim(-4x/x) = e**-4
    x→∞                        x→∞                    x→∞
Неопределенности типа 0/0
    lim  2x²-3x-5   0   (x+1)*(2x-5)
    x→-1 -------- = - =  ----------- = -7
         x+1        0        x+1
    нужно разложить числитель и знаменатель на множители
    сократить лишнее и подставить в x число под lim

Правило Лопиталля
    решает неопределенности типа 0/0 ∞/∞
    1 правило:
        рассмотрим  две фу-ии, которые бесконечно малы в некой точке k
        lim f(x)/g(x) = 0/0 # неопред-ть
        x→k
        lim f'(x)/g'(x)     # возьмем производные от этих ф-ий
        x→k                   и получим решение подставив k
    2 правило:
        рассмотрим  две фу-ии, которые бесконечно большие в некой точке k
        lim f(x)/g(x) = ∞/∞ # неопред-ть
        x→k
        lim f'(x)/g'(x)     # возьмем производные от этих ф-ий
        x→k                   и получим решение подставив k
        
1 замечательный предел:
    lim sinx/x = 1
    x→0
2 замечательный предел:
    lim (1+1/x)**x = e = 2.72
    x→∞

Бесконечно малые функции
    Если сущ предел lim f(x) = 0 - то ф-я f(x) наз-ся бесконеч малой в точке k
                    x→k
    Функция может быть беск малой лишь в конкретной точке
    f(x) = x    # функция

    x→0

Производная -------------------------------------------------------------------
Дифференцирование - нахождение производной.
Чтобы найти производную функции нужно по определенным правилам превратить ее 
в другую функцию.

Обозначения: f'(x), y', dy/dx.

Правила дифференцирования:
1. Постоянное число можно (и нужно) вынести за знак производной
    (Cu)' = Cu' # C - константа
    y = 3cosx => y' = (3cosx)' = 3(cosx)' = 3(-sinx) = -3sinx
2. Производная суммы равна сумме производных
    (u ± v)' = u' ± v'
    y = 6 + 4x => y' = 6' + 4x' = 0 + 4 = 4
3. Производная произведения функций
    (uv)' = u'v + uv'
    y = x²*sinx => (x²)'*sinx + x²*(sinx)' = 2x*sinx + x²*cosx
4. Производная частного функций
    (u/v)' = (u'v-uv')/v²
    y = (2x+1)/5x => y' = ((2x+1)'*5x - (2x+1)*(5x)')/(5x)² = -1/5x²
5. Производная сложной функции
    (u(v))' = u'(v) * v'
    u(v) - функция в функции => sin(2x) => 2x - внутренняя, sin() - внешняя
    Найдем производную y' = (sin(2x))'
    u'v = (sin(2x))' = cos(2x)  # находим производную внешней ф-ии
    v' = (2x)' = 2              # затем внутреннюю
    y' = 2cos(2x)               # подставим в формулу, получим
6. Логарифмическая производная
    y = (x+4)**3/(x-1)**4
    ln(y) = ln((x+4)**3/(x-1)**4)       # навешиваем логарифмы
    ln(y) = ln((x+4)**3) - ln((x-1)**4) # исп свойства логарифмов(деление)
    ln(y) = 3*ln(x+4) - 4*ln(x-1)       # исп свойства логарифмов(степень)
    (ln(y))' = (3*ln(x+4) - 4*ln(x-1))' # берем производную от обеих сторон
    1/y * y' = 3/(x+4) - 4/(x-1)        # в левой части слож ф-я поэтому так
    y' = (3/(x+4) - 4/(x-1))*y          # ответ y - изнач ф-ия
7. Производная степенно-показательной ф-ии
    y = x**x
    ln(y) = ln(x**x)        # вешаем логарифмы
    ln(y) = x*ln(x)         # исп свойства логарифмов(степень)
    (ln(y))' = (x*ln(x))'   # берем производную от обеих сторон
    1/y * y' = (x)'*ln(x) + x*(ln(x)')  # как в примере выше
    y' = (ln(x)+1) * x**x   # решение
8. Производная функции, заданной неявно
    3x²y² - 5x + siny = 3y - 1  # неявная ф-ия, невозм выразить переменную
    (3x²y²-5x+siny)' = (3y-1)'  # берем производные
    (y)' = y'                   # сама производная
    (siny)'= y'*cosy            # как сложн ф-ию
    y²  # все y с наворотами - сложные ф-ии
    ...
    y' = (5-6xy²)/(6x²y+cosy-3) # ответ

Производная функции в точке:
    f(x) = 2x² + 2  # функция
    x0 = 2          # точка
    f'(x) = 4x      # находим производную ф-ии
    f'(x0) = 8      # подставляем значение заданной точки в x

Уравнение касательной к графику функции:
    f(x) = 2x² + 2              # функция
    x0 = 2                      # точка по Х
    y - f(x0) = f'(x0)*(x-x0)   # общ формула 
    f(x0) = 10                  # чему равна ф-ия в точке (по Y)
    f'(x) = 4x                  # производная ф-ии
    f'(x0) = 8                  # производная в точке
    y - 8x + 6 = 0              # подставим в формулу -> получим ур-е касат
    10 - 8*2 + 6 = 0            # проверим точку (x0;f(x0)) = (2;10), подставим
    0 = 0                       # равенство верно

Уравнение нормали к графику ф-ии:
    нормаль - прямая прох через точку касательной, перпендикулярно касательной
    y = f(x)                    # function
    (x0;f(x0))                  # point
    y-f(x0) = -1/f'(x0)*(x-x0)  # уравнение

Дифференциал функции одной переменной:
    dy = y'*dx либо df(x) = f'(x)*dx    # обозначение
    y = 2x + 3  # функция
    y' = 2      # найти производную
    dy = 2dx    # ответ

Приближ вычисления с пом дифференциала ф-ии 1-ой переменной:
    67**(-1/3)          # вычислить приближенно 67 в корне кубическом
    f(x0+Δx) = f(x0)+d[f(x0)]   # рабочая формула
    f(x) = x**(1/3)     # вид ф-ии
    f(67) = 67**(1/3)   # подставляем наше значение
    67 -> x0+Δx         # 67 необходимо разбить на x0 и Δx
    x0 = 64             # находим ближ целое число в кубе 
    Δx = 67-64 = 3      # это приращение аргумента
    f(x0) = f(64) = 4   # подставляем, находим
    d[f(x0)]= f'(x0)*Δx # дифференциал в точке из формулы с производной
    f'(x)= 1/3*x**(2/3) # находим производную ф-ии
    f'(x0) = 1/48       # подставляем наше знач 64, нах произв в точке
    d[f(64)] = 1/16     # подставл и находим дифф в точке
    f(67) = 4+1/16      # находим приблизит значение

Приближ вычисления с пом дифференциала ф-ии 2-х перем:
    f(x0+Δx,y0+Δy) = f(x0,y0) + d[f(x0,y0)] # рабочая формула
    алгоритм решения такой же

Геометрич смысл производной в точке:
          |Y
          |                     a,b     - экстремумы ф-ии
          |    y = f(x)         Δx      - некое расстояние по X
       _  |             _       Δy      - расст по Y соответсв Δx по ф-ии y
      / \ +f(x0+Δx)  L_/        Δy/Δx   - коэф. крутизны ф-ии, чем выше, тем
     /   \|       __/|                    круче график, положителен при воз-
    /     +f(x0)_/___| Δy                 растан ф-ии, при убывании - отриц,
   /      |\   / K   N                    нулевой при ровной ф-ии и экстрем-х
  /       | \_/    Δx      X    
-/-----+--+--+---+---+-----     KL      - секущая
/      a  |0 b   x0  Δx+x       KNL     - прямоуг треугольник
          |                     ∠LNK(α) - уг накл секущей к 0X
выводим формулы:
    Δy = f(x0+Δx)-f(x0) # приращение функции
    tgα = LN/KN = Δy/Δx # тангенс угла секущей к 0X
определение производной:
    Производной функции в точке x0 называется предел отношения приращения
    функции Δy к вызвавшему его приращению аргумента Δx в этой точке при Δx→0. 
    +-----------------------------------------+
    |              f(x0+Δx) - f(x0)        Δy |
    | f'(x0) = lim ---------------- = lim --- |
    | Δx→0               Δx           Δx→0 Δx |
    +-----------------------------------------+
    Секущая KL при Δx→0 стремится занять положение касательной к графику 
    функции в точке x0 (точка K)
    Касательная к графику ф-ии в точке это предельное полож секущей в этой точ.
    Угол наклона секущей α стремиться к углу наклона кательной к 0X - φ.
    +---------------------------+
    | Δy               Δy       |   Производная ф-ии в точке x0 
    | -- = tgα => lim --- = tgφ |   численно равна тангенсу угла наклона
    | Δx          Δx→0 Δx       |   касательной к графику ф-ии в этой точке
    +---------------------------+

Дифференциал функции в точке и его геометрический смысл:
    Дифференциалом функции в точке x0 называют главную линейную часть приращения
    функции Δy.
обозначения
    dy, d[y(x0)], d[f(x0)]
               L
     ---------+     dy = NM - дифференциал, при бесконечно малом Δx dy→Δy
           __/|
        __/  M+ Δy  f'(x0) = d[f(x0)] / dx  # dx - предельно малое Δx 
       /    dy|
    K +-------+ N
          Δx

Понятие производной функции:
    Из формулы производной в точке вместо x0 можно подставить любое x принадл
    области опред ф-ии, в котором сущ производная(быв не сущ типа деление на 0)
    +--------------------------------------+
    |             f(x+Δx) - f(x)        Δy | из этой формулы появилась вся
    | f'(x) = lim -------------- = lim --- | таблица производных и правила
    | Δx→0              Δx         Δx→0 Δx | дифференцирования
    +--------------------------------------+   <--+
    Итог: для функции y = f(x) по закону (формула-+) ставится в соответствие
    другая функция y' = f'(x), которая является производной функцией!
    Производная характеризует скорость изменения ф-ии, если
        f'(x0) > 0  - функция возрастает в этой точке
        f'(x0) < 0  - функция убывает в этой точке
        f'(x0) = 0  - сохраняет свою скорость постоянной
    Дифференцировать ф-ию - значит выявить ее скор изменения в виде производной

пример:
    f(x) = x² + 2   # функция
    f'(x) = ?       # найти производную по опрделению(табл выше)
    x0              # рассмотрим произвольную точку
    f(x0) = (x0)²+2 # значение функции в x0
    Δx              # зададим приращение аргумента
    x0+Δx           # точка стоящая на расст Δx
    f(x0+Δx) = (x0+Δx)²+2 = (x0)²+2x0Δx+(Δx)²+2 # знач функции в точке x0+Δx
    Δy = f(x0+Δx) - f(x0)   # приращение функции
    Δy = 2x0Δx + (Δx)²      # подставив получим
                  Δx       2x0Δx+(Δx)²                          # по определению
    f'(x0) = lim --- = lim ----------- = lim(2x0+Δx→0) = 2x0      производной
             Δx→0 Δy  Δx→0    Δx         Δx→0                     в точке
    f'(x) = 2x      # в кач-ве x0 можно рассм люб точку x области опред функции
                      f(x) = x² + 2, поэтому получим f'(x) = 2x
    Подставим несколько точек в производную
    f'(-0.5) = -1   # тут функция медленно убывает
    f'(0) = 0       # тут сохраняет скорость
    f'(1) = 2       # возрастает
    f'(2) = 4       # круто возрастает в точке x = 2

    Δ = |точное значение - приближенное знач|   # абсолютная погрешность
    σ = Δ/точн.знач * 100%                      # относительная погрешность

Первообразная функция
Это задача обратная задаче найти производную
Интегрирование функции - нахождение ее первообразной/интеграла
f(x)  = x**3       # обычная заданная функция
f'(x) = 3*x**2     # производная
F(x)  = x**4/4     # первообразная - это функция производная которой равна заданной ф-ии
F(x)  = x**4/4 + 1 # также первообразная для f(x) = x**3
F(x)  = x**4/4 + с # в общем + любое число(константа),
                     может быть бесконечное кол-во перво-х(это и есть интеграл)
F'(x) = f(x)       # производная от первообразной - это исходная функция

Неопределенный интеграл 
Существует елси ф-я непрерывна на промежутке Х
Это множество всех первообразных для ф-ии f(x) на некотором промежутке X
Геометрически это семейство одинаковых интегральных кривых идущих вдоль OY (F(x)+c) c-сдвиг
∫f(x)*dx = F(x) + c   # интеграл от ф-ии f(x)dx
                        ∫      - неопределенный интеграл
                        f(x)   - подынтегральная ф-ия
                        f(x)dx - подынтегральное выражение
                        dx     - дифференциал аргумента(Δx)
                        x      - переменная интегрирования
                        F(x)   - первообразная f(x)
                        c      - константа(люб число)
1 свойство:
d(∫f(x)*dx) = d(F(x) + c) = dF(x) + dx = F'(x)*dx + 0 = f(x)*dx
(∫f(x)*dx)' = (F(x) + c)' = F'(x) + 0 = f(x)
2 свойство:
∫dF(x) = ∫F'(x)*dx = ∫f(x)*dx = F(x) + c
3 свойство:
∫a*f(x)*dx = a*∫f (x)*dx

Определенный интеграл
b          N
∫f(x)*dx = Σ f(xi)*∆xi = f(x1)*∆x1 +f(x2)*∆x2 +...+f(xi)*∆xi
а         i=1       
∆x - длина отрезков по OХ(бесконеч множество между а и b)
f(x) - значение по Y, х берется произвольно на отрезке ∆х
Интеграл это сумма произведений отрезков ∆х на значения f(x) , стремящихся к бесконечности.
Площадь под графиком до оси OX - это и есть эта сумма произведений.
