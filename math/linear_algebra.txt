Linear Algebra
Раздел алгебры, изучающий объекты линейной природы: векторные (или линейные)
пространства, линейные отображения, системы линейных уравнений, среди основных
инструментов, используемых в линейной алгебре — определители, матрицы,
сопряжение. Теория инвариантов и тензорное исчисление.


Вектор------------------------------------------------------------------------
(геом) направленный отрезок, для которого указано его начало и конец
(мат)  в простейшем случае мат объект, харак-ся величиной и направлением.
       упорядоченная последовательность действительных чисел a1,a2,a3,…,an-1,an
AB      # вектор, А - начало, В - конец
|AB|    # модуль вектора - его длина
# коллинеарные вектора - || одной прямой или лежащие на одной прямой
# сонаправленные вектора, коллинеарные + их направления совпадают
# противоположно напр вектора - коллинеарные + напр противоположны
# компланарные Вектора - || одной плоскости или лежащие на одной плоскости
# Всегда возможно найти плоскости параллельную двум произвольным векторам,
# по этому любые два вектора всегда компланарные.
# Вектора a и b называются равными, если они лежат на одной или || прямых,
# их направления совпадают, а длины равны
# Вектора a и b называются ортогональными, если угол между ними равен 90°(⊥),
# если их скалярное произведение равно нулю
# базис вектора - образуют базис(основу) для изображения векторов
# Базис плоскости - пара линейно независимых (неколлинеарных) векторов,
    взятых в определённом порядке, при этом любой вектор плоскости является
    линейной комбинацией базисных векторов.
i - единичный вектор по X
j - единичный вектор по Y
O - начало координат
i ⊥ j # ортогональны

  |
 j|___  вектор строим откладкой сначала по i потом по j
 O i 
 
# Разложение вектора по базису(i;j)
v = v1*i + v2*j # v1, v2 - координаты вектора v, i,j - базисы
# Координаты точек   – это обычные координаты в прямоугольной системе координат
# Координаты вектора – это его разложение по базису
v = (x1-x2;y1-y2) = (A*i;B*j) # найти координаты вектора по координатам точек
|v| = √((x2-x1)² + (y2-y1)²)  # найти модуль вектора через координаты точек
a + b = (a1;a2) + (b1;b2) = (a1+b1;a2+b2) # сумма векторов
a - b = (a1;a2) - (b1;b2) = (a1-b1;a2-b2) # разность векторов
0° <= ∠(a;b) <= 180°(π)       # угол между векторами
a * b = |a| * |b| * cos∠(a;b) # скал произв векторов(при ∠(a;b) = 90° => 0)
a * b = a1*b1 + a2*b2         # скал произв векторов зад в ортонорм базисе
[a*b] = |a| * |b| * sin∠(a;b) # вект произв векторов(вектор = S параллелогр-ма)
a² = |a|²
|a| = √a²
cos∠(a;b) = (a1*b1 + a2*b2) / √(a1²+a2²) * √(b1²+b2²)   # выражаем угол
# проекции вектора на оси X и Y - это косинусы угла между вектором и осью

# Сложение/вычитание матриц
    Складывать и вычитать можно матрицы одного размера в результате 
    получается матрица того же размера. Необходимо сложить соответствующие

# умножение матриц
      K       L    # необходимо чтобы кол-во столбцов K == кол-ву строк L
    a1 b1 * c1 d1  # a1c1+b1c2 a1d1+b1d2
    a2 b2   c2 d2  # a2c1+b2c2 a2d1+b2d2
    # переставлять местами матрицы нельзя!
# обратная матрица
    A**(-1) = 1/|A| * At
    обратная матрица = 1/определитель матрицы А и все это * на транспонир мат А
    если определитель матрицы = 0, то обратной матрицы не сущ(смотри лин преобр)
# Детерминант - определитель матрицы
    это скалярная величина, которая обозн ориентированное «растяжение» или
    «сжатие» многомер-го евклидового простр-ва после преобразования матрицей
    имеет смысл только для квадратных матриц. Допустим в двумерном простр
    площадь S квадратика между базовымии i(1;0) и j(0;1) = 1, так вот если
    применить линейное преобразование(допустим растяжение i(2;0)) - 
    получим площадь прямоугольника между получившимися базисами будет в 2
    раза больше, так вот Детерминант это скалярная величина(коэф),
    показывающ на сколько изменилась полощадь - в данном случае 2. Если
    получается отрицательное число - ориентация пространства поменялась
    (либо i стал направлен влево, либо j вниз)

# Свойства определителя
    - При транспонировании матрицы определитель не меняется
    - Если две строки (или два столбца) определителя поменять местами,
        то определитель сменит знак
    - Из строки (столбца) определителя можно вынести общий множитель
    - Если две строки (столбца) определителя пропорциональны (как частный
        случай – одинаковы), то данный определитель равен нулю
    - Определитель с нулевой строкой (столбцом) равен нулю


Системы линейных уравнений----------------------------------------------------
уравнения с неизвестными не выше первой степени
Решить систему уравнений – это значит найти множество её решений. Решение
    системы представляет собой набор значений всех входящих в неё переменных,
    который обращает КАЖДОЕ уравнение системы в верное равенство. Кроме того,
    система может быть несовместной (не иметь решений)
решение:
    x - y = -5  
    2x + y = -7
    метод подстановки:
    x = y - 5       # выражаем в одном из уравнений одну из переменных
    2*(y-5) + y = 7 # подставляем во второе
    y = 1           # находим y
    x = -4          # подставляем y в любое и находим x

    метод почленного сложения(вычитания) уравнений системы:
    3x = -12        # суть метода избавиться от одной переменной складывая
    x = -4, y = 1   # или вычитая уравнения, также можно умножать любое
                    # уравнение на число чтобы подвести ко второму
    метод Крамера:
    a1x+b1y = c1
    a2x+a2y = c2
    D  a1 b1        # найти главный определитель, если = 0, то решений
       a2 b2         либо беск множ-во, либо нет
    Dx c1 b1        # найти следущ определитель 
       c2 b2
    Dy a1 c1        # и еще один
       a2 c2
    x=Dx/D y=Dy/D   # нахождение корней
    x - y = -5  -> D 1 -1 = 3 Dx=-12 Dy=3 -> x=-4; y=1
    2x + y = -7      2  1

    матричный метод:
                   A    x   v
                  ----------- 
    x - y = -5    1 -1  x  -5 -> x = A**(-1) * v # обратная матрица А * v
    2x + y = -7   2  1  y  -7

    метод Гаусса:
    1 -1|-5         # запишем расширенную матрицу системы
    2  1|-7           вертикальная черта для видимости(не имеет знач)
    2  1|-7         # строки можно переставлять местами
    1 -1|-5         # если есть дубликаты или пропорциональные строки
                      их можно убирать, оставить только одну оригинальную строку
                    # можно убирать нулевые строки
    2 -2|-10        # строку можно умножать/делить на люб число не равное 0
    0  3|3          # можно прибавлять другую строку умнож на число
                      ко второй прибавил первую * -2: (1 -1 -5)*(-2) -2 3 10
                      в итоге получилось 0 3 3
    1 -1|-5         # Цель элементарных преобразований – привести матрицу к
    0  1|1            ступенчатому(треугольному) виду(все что ниже гл диаг=0)
    x - y = -5      # в итоге раскручиваем все обратно - обратный ход Гаусса
        y = 1         все находится элементарно для любых уравнений
                    # при решении системы первым ставим уравнение у которого
                      первый коэф = 1 или -1
    Метод Гаусса-Жордана:
        похож на метод Гаусса только приводим матрицу к верхнетреугольной

Несовместные системы и ситемы с беконечным числом решений
    кол-во уравнений меньше чем кол-во неизвестных - система несовместна либо
    имеет беск число решений
    если в итоге преобразований к супенч виду получилась система вида:
        -1 0 1 3|7
         0 4 2 9|14
         0 0 0 0|2  - если есть такое - система не имеет решений

         2 3 -1 1|1 
         0 0 5 -4|1 - система имеет бесконечное множ-во решений
         перепишем
         2x1 + 3x2 -x3 + x4 = 1  найдем базисные переменные - они находятся
                  5x3 - 4x4 = 1  на ступеньках - т.е. x1 и x3 (крайние в ур-ии)
                                 свободные переменные - те что остались x2 x4
                                 теперь выражаем все базисные через свободные
         решение
         x1 = -3x2/2 - x4/10 + 3/5
         x3 = 4x4/5 + 1/5
         передавая х2 и х4 произвольные значения находим беск множ решений
         подставим х2=0 х4=0
         (3/5;0;1/5;0) - частное решение
         потом подставляем в исходную систему(до элем преобразований) - проверка

Ранг матрицы
    Ранг матрицы - порядок базисного минора.
    Ранг = макс число линейно независимых строк (столбцов).
    Метод Гаусса - выполняем элем преобразования до конца и находим
    кол-во линейно независимых строк - это и есть ранг матрицы
        1 -1  2     ко второй прибавляем 1-ую *2
        2 -2  4 ->  к 3-ей прибавим 1-ую        -> 1 -1 2 -> ранг=1
        -1 1 -2

Линейные преобразования
# Все линейные преобразования осуществляются с помощью базисных векторов
    используя их координаты можно совершить любое линейное преобразование
    (повороты, наклоны, растяжения) как бы меняем базис и все что в простран-ве
    меняется также.

     |i j # базисные вектора        v # произвольный вектор
    -+---                           -
    x|1 0 # координаты базисных     2 # координаты(2;3)(0;0 2;3)
    y|0 1   векторов по умолчанию   3
 
# Базисные вектора удобно записывать в матрицу 
# Линейное преобразование - поворот на 90 градусов
    вектор      матрица     результат
    v 2     ->  0 -1    ->  v -3    
      3         1  0           2
    векторно-матричное перемножение
    0 -1   2 = 0*2+(-1)*3 = -3
    1  0 * 3   1*2+0*3       2
# 2 линейных преобразования - это произведение матриц
    поворот   наклон    итоговая матрица
     0 -1      1 1    0*1-1*0 0*1-1*1  0 -1
     1  0      0 1    1*1+0*0 1*1+0*1  1  1
     преобразование идет справа налево наклон -> затем поворот!

# Системы линейных уравнений
    система линейных уравнений это матрица коэф умноженная на 
    вектор неизвестных равная некоторому вектору констант
                       A     x   v
    ------------------------------
    2x+5y+3z = -3    2 5 3   x  -3
    4x+0y+8z = 0  -> 4 0 8 * y = 0  -> Ax = v # v это x после преобразования A
    1x+3y+0z = 2     1 3 0   z   2              чтобы найти x необходимо обрат
                                                преобраз для v -> x = A**(-1)v

# Собственный вектор
    Собственный вектор матрицы A - ненулевой вектор v, который при умножении 
    на некоторую квадратную матрицу A превращается в самого же себя 
    с числовым коэффициентом λ - собственное число матрицы
        -1 -6
    A =  2  6   # дана матрица A, найти собственные числа и векторы
        x
    u = y       # собственный вектор, кот нужно найти
    Au = λu     # применение матрицы к вектору = собств число * этот вектор
    # подставляем
        -1 -6   x      x         -x-6y   λx     -x-6y = λx      -x-6y-λx = 0
        2  6  * y = λ* y         2x+6y = λy     2x+6y = λy      2x+6y-λy = 0
    # вынесем x(в первом) и y(во втором) за скобки
        (-1-λ)x-6y = 0
        2x+(6-λ)y = 0
    # По определению, собственный вектор не может быть нулевым, x=0 y=0 не
        устраивает => уравнения линейно зависимы и D матрицы системы = 0
    # характеристическое уравнение матрицы - корни - собственные числа
       -1-λ  -6         # вычитаем из глав диаг собств число
    Det 2    6-λ = 0    
    # найдем определитель приравняем к 0 и решим квадр урав
        (-1-λ)*(6-λ)-2*(-6) = 0
        λ² - 5λ + 6 = 0
        λ1 = 2; λ2 = 3  # это собственные числа
    # найдем собственные векторы, подставим по очереди λ1 λ2 в однород сист ур
        (-1-λ)x-6y = 0  -3x-6y = 0  
        2x+(6-λ)y = 0    2x+4y = 0  =>  x = -2y # бесконечно мног собс векторов
    # Все они будут коллинеарны друг другу, достаточно указать один из них
    # чтобы его «иксовая» координата была положительной, целой и минимальной, 
        а «игрек» не дробным
        y = -1
        x = 2
    # находим второй вектор по второму собс числу
        y = -2
        x = 3
    # ответ
        собственные числа: 2, 3
        собственные векторы: (2;-1), (3;-2)
