Прямая пропорциональность
    y = k*x (k !=0) # прямая пропорциональная зависимость
    k - коэф пропорциональности или угловой коэф
    График - прямая проходящая через О(0,0)
    k>0 - график в I and III четвертях
    k<0 - график в II and IV

Обратная пропорциональность
    y = k/x (k!=0) # обратная пропорциональная зависимость
    k - коэф обр пропорциональности
    График - гипербола, состоит из 2-х ветвей, симметричен относительно О(0,0)
    k>0 - график в I and III четвертях
    k<0 - график в II and IV

Преобразования графика:
    F(x) = f(x) + b - перенос графика f(x) на вектор (0;b) вдоль OY
    F(x) = f(x + а) - перенос графика f(x) на вектор (-a;0) вдоль OX
    F(x) = f(x + a) +  b- перенос графика f(x) на вектор (-a;b)
    F(x) = -f(x) - симметричное отражение графика f(x) относительно оси OX
    F(х) = A*f (x), 0 < А < 1 Сжатие графика f(х) в A раз вдоль оси OY
    F(x)=A*f(x),A>l Растяжение графика f(х) в А раз вдоль оси OY
    F(x) =f(-x) Симметричное отображение графика f(х) относительно оси ОУ
    F(х) = f(k*x), 0 < k < 1 Растяжение графика f(х) 1 в 1/k  раз вдоль оси ОХ
    F(x) = f(k*x), k > 1 Сжатие графика f(х) в k раз вдоль оси ОХ

Общая схема построения графика
    y = m*f(k*x + b) + h
    k - коэф растяжения вдоль OX, если отрицательный, отражаем график.
    b - сдвиг по OX
    m - коэф растяжения вдоль OY
    h - сдвиг по OY

Линейная функция
    y = k*x + b # прямая
    k # угловой коэф(влияет на наклон прямой)
    b # сдвиг по оси OY
    График - прямая проходящая через (0,b)

Общее уравнение прямой
    a*x + b*y + c = 0 # можно создать любую прямую, даже со сдвигом по OX и OY

Квадратичная функция
    y = a*x² + b*x + c
    a != 0, b, c - действительные числа
    x - независимая переменная
    область определения - все действ числа x in R
    нули квадр ф-ии - значения х при которых у==0
    y = x² - парабола (a=1, b=c=0)
    y = a*x² - при  а>1 - парабола сужается, 0<а<1 - расширяется
    y = a*x² + y0   - сдвиг вершины по оси Y на у0
    y = a*(x - x0)² - сдвиг вершины по оси Х на х0
    График - парабола
    y = a*x² + b*x + c
    y = a*(x - x0)² +y0
    x0 = -b/2*a # сдвиг по Х
    y0 = -(b² - 4*a*c)/4*a # сдвиг по Y
    Найти вершину
    берем производную, риравниваем к нулю - находим корень(x)
    Далее подставляем в уравнение(функцию) - находим y

Функция f(x) = √x
    область определения D(y) = [0;+∞)
    График - ветвь параболы, которая лежит на боку

Функции f(x) = ³√x, f(x) = x³
    область определения = R
    проходят через (0;0)
    График - кубическая парабола ,³√x лежит на боку

Функция f(x) = a/x
    D(y) = все кроме 0
    График - гипербола

Показательная функция(степенная)
    y = a**x # a>0, a!=1
    2 случая:
    0 < a < 1 # (1/2)**x
    a > 1     # 2**x
    D(y) = R
    E(y) = (0;+∞)
    (0;1) — характерная точка

логарифмическая ф-ия
    это обратная степенной функции
    y = logaX # a>0, a!=1
    D(y) = (0;+∞)
    E(y) = R
    (0;1) — характерная точка

Тригонометрия ---------------------------------------------------------------
Геометрический угол - фигура образованная двумя лучами, выходящими из 1 точки - вершина угла.
Полный ∠ - 360° - 2*π
Развернутый ∠ - прямая - 180° - π
Прямой ∠ - половина развернутого - 90° - π/2
1° - развернутый ∠/180
1' = 1°/60
1" = 1'/60
Центральный угол - угол с вершиной в центре окружности
1 радиан = центральный угол с длиной дуги == R окружности
Радианная мера — угловая мера, в которой за единицу принимается угол в 1 радиан.
То есть, радианная мера любого угла — это отношение этого угла к радиану. 
радианная мера угла — отношение длины дуги окружности, находящейся между сторонами
угла, к радиусу этой окружности, когда центр окружности совпадает с вершиной угла. 
1° = 0.01745 рад
1 рад = 57° 17'45"

Град 30° 45° 60° 90° 180° 270°  360°
Рад  π/6 π4  π/3 π/2  π  3*π/2  2*π

π = длина окружности/диаметр = 3.14
Длина окружности = π * диаметр
(0;0) - вершина угла
Ось Х совпадает с одной из сторон треугольника
а - угол относит оси Х и второй стороны, образующей угол
Для любой точки на окружности существуют х;у координаты
х - это cos a
y - it's sin a
Соответственно если угол 0° то у точки будут координаты R;0
0° (R;0) cos a = R, sin a = 0
45° cos a = sin a 
90° (0;R) cos a = 0, sin a = R
180° (-R;0) cos a = -R, sin a = 0
270° (0;-R) cos a = 0, sin a = -R

формулы по прямоуг треугольнику
sin a = y/R
cos a = x/R
tg a = y/x
ctg a = x/y
sc a = 1/cos a
csc a = 1/sin a

формулы между собой
sin² a + cos² a = 1
tg a = sin a/ cos a = 1/ctg a
ctg a= cos a/ sin a = 1/tg a
tg a * ctg a = 1
1 + tg² a= 1/cos² a
1 + ctg² a = 1/sin² a

Обратные тригонометрические функции
арксинус, арккосинус, арктангенс, арккотангенс
sin π/6 = 1/2
arcsin 1/2 = π/6

Гармонические колебания
y = A* sin(w*x +f)
w - циклическая частота, влияет на растяжение вдоль ОХ
f - начальная фаза колебания, сдвиг по оси ОХ
А - амплитуда колебаний, растяжение вдоль ОУ
T = 2*π/w - период гармонического колебания
Изменение величины у происходит по закону:
y = A* sin(w*t +f) либо y = A* cos(w*t +f)
t - независимая переменная (время)

Решение треугольников
a,b,c - стороны треугольника
A,B,C - противолежащие им углы
R  - радиус описанной окружности   
r  - радиус вписанной окружности
S  - площадь
p  - (a+b+c)/2 полупериметр
ha - высота опущ на сторону a
la - биссектриса угла A

теорема синусов
a/sinA = b/sinB = c/sinC = 2*R
теорема косинусов
a² = b² + c² - 2*b*c * cosA
площадь
S = b/2 * c * sinA
радиусы
R = a/(2*sinA)
r = S/p
    
--- Формулы деления отрезка в данном отношении на плоскости-----------------
A(xa;ya), B(xb;yb)       # известные точки плоскости
M(xm;ym)                 # точка делящаяя отрезок в отношении AM/MB‎
λ = ‎AM/MB                # отношение отрезков
xm = (xa + λ*xb)/(1 + λ) # x coord of M
ym = (ya + λ*yb)/(1 + λ) # y coord of M
zm = (za + λ*zb)/(1 + λ) # если 3д пространство
середина отрезка по 2-м точкам
xm = (xa + xb)/2, ym = (ya + yb)/2 # M - середина отрезка

--- Уравнение прямой на плоскости------------------------------------------------
y = k*x + b         # уравнение с угловым коэф k
y - y0 = k*(x - x0) # при известной точке M(x0;y0) и уг коэф k (выражаем y через x)

Общее уравнение прямой
Ax + By + C = 0 # A,B,C - некоторые числа(A & B одноврем != 0)

уравнение прямой по двум точкам
M1(x1;y1), M2(x2;y2)            # известные точки
(x-x1)/(x2-x1) = (y-y1)/(y2-y1) # уравнение

Направляющий вектор прямой - вектор, который параллелен прямой
уравнение прямой по точке и направляющему вектору
M(x0;y0) # точка принадлежащая прямой
p(p1;p2) # направляющий вектор прямой(p1 != 0 и p2 != 0)
(x - x0)/p1 = (y - y0)/p2 # каноническое уравнение прямой
Ax + By + C = 0           # прямая заданная уравнением
p(-B;A)                   # направляющий вектор данной прямой

Вектор нормали прямой (нормальный вектор)
нормаль - перпендикуляр прямой
Вектор нормали всегда ортогонален направляющему вектору прямой
уравнение прямой по точке и вектору нормали
M(x0;y0) # точка принадлежащая прямой
n(n1;n2) # вектор нормали данной прямой
n1*(x-x0)+n2*(y-y0) = 0 # уравнение
Ax + By + C = 0 # прямая заданная уравнением
n(A;B)          # вектор нормали данной прямой
