Start using Linux
who             - show who is logged on
w               - show who is logged on and what they are doing
hostname        - show or set the system's host name
passwd          - change user password
date            - print or set the system date and time
cal             - displays a calendar and the date of Easter
exit            - cause normal process termination
poweroff        - power-off or reboot the machine

Directories
ls              - list directory contents
mkdir <path>    - make directories
pwd             - print name of current/working directory
cd              - change directory
rmdir <path>    - remove empty directories
find            - search for files in a directory hierarchy

Files
ls              - list directory contents
touch <path>    - change file timestamps/new file
rm <path>       - remove files or directories
wget <url>      - the non-interactive network downloader
file <path>     - determine file type
cp              - copy files and directories
mv              - move (rename) files

File contents
head <path>     - output the first part of files
tail <path>     - output the last part of files.
cat <path>      - concatenate files and print on the standard output
tac <path>      - concatenate and print files in reverse
wc <path>       - print newline, word, and byte counts for each file
grep <path>     - print lines that match patterns
more <path>     - file perusal filter for crt viewing
echo 'str'      - display a line of text
