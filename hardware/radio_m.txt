Радиоволны 
    электромагнитные волны с частотами от 30 кГц до 3000 ГГц(радиочастоты)
    соответствует длине волны от 10 километров до 0,1 миллиметра

Распространение волн
        Название                        Распространение
    --------------------------------------------------------------------------------------------------------------------
    1   Прямые                          свободном простр от одного предмета к другому без помех
    2   Земные/поверхностные            вдоль сферической поверхности Земли и частично огибающие её вследствие явления дифракции
    3   Тропосферные                    ОВЧ и УВЧ, распространяющиеся за счёт рассеяния на неоднородностях тропосферы на расстояние до 1000 км
    4   Ионосферные/пространственные    радиоволны длиннее 10 м, распространяющиеся вокруг земного шара на сколь угодно большие расстояния за счёт однократного или многократного отражения от ионосферы и поверхности Земли
    5   Направляемые                    в направляющих системах (радиоволноводах)


Начальное оборудование
SDR:        RTL-SDR Blog V4
антенна:    провод витая пара распутанная 2 жилы 20м + леска с грузом + SMA-female разъём
софт:       SDR++
USB удлиннитель

=== SDR ==============================================================================================
зависимости:
    sudo apt update
    sudo apt install -y git cmake build-essential qtbase5-dev qtchooser qt5-qmake qtbase5-dev-tools libgl1-mesa-dev libfftw3-dev libvolk2-dev libsoapysdr-dev libairspy-dev libairspyhf-dev libhackrf-dev libbladerf-dev

драйвер:
    sudo apt install -y rtl-sdr

убрать из автозагрузки:
    sudo bash -c 'echo "blacklist dvb_usb_rtl28xxu" >> /etc/modprobe.d/blacklist-rtl.conf'
    sudo bash -c 'echo "blacklist rtl2832" >> /etc/modprobe.d/blacklist-rtl.conf'
    sudo bash -c 'echo "blacklist rtl2830" >> /etc/modprobe.d/blacklist-rtl.conf'
    sudo reboot

скачивание-сборка:
    git clone https://github.com/AlexandreRouma/SDRPlusPlus.git
    cd SDRPlusPlus
    mkdir build
    cd build
    cmake ..
    make -j$(nproc)

запуск
    ./sdrpp

Настройка SDR++ для приёма:
    Устройство:выберать RTL-SDR
    Частота: вводи в МГц
    Модуляция: выбери AM, NFM, USB, LSB, CW и т.д.
    Waterfall (водопад): можно увеличить FFT и масштаб

Полезные опции:
    Автоматический HF-режим на V4 включён по умолчанию.
    Bias-T (если используешь внешние антенны с питанием) можно включить в настройках устройства.
    Запись IQ или WAV: через иконку записи

Проверка работы после запуска:
    rtl_test

    Found 1 device(s):
    0:  Realtek, RTL2838UHIDIR, SN: 00000001

    Using device 0: Generic RTL2832U OEM

Другое ПО(позже если нужно)
    Fldigi (для цифровых сигналов): sudo apt install fldigi
    Gqrx (ещё один SDR GUI): sudo apt install gqrx-sdr
    GNU Radio: мощный фреймворк, но сложный: sudo apt install gnuradio


=== Антенна ===============================================================================
Для КВ (HF, 3–30 МГц):
    Это самый интересный диапазон: DX-приём, радиолюбители, вещание, авиация и др.
    Антенна типа "проволока за окно":
    Берём 10–20 метров медного провода, кидаем за окно (можно на шнурке, грузике).
    Один конец втыкается в балун (например, 9:1 UnUn), или просто через вход RTL-SDR через трансформатор или адаптер.

    Рабочий вариант:
        Жилы 2 не зачищенных из витой пары 20м(цвет незаметный)
        Распутать выровнять(1-сигнал, 2-земля)
        Леска 0.8 мм - 1.5 мм
        Изолента черная
        Груз 300 гр с петлёй/хомутом мягкий незаметный
        Примотать изолентой провода к леске на каждых 15 см (по 2-3 витка), оставить снизу чуть больше лески
        Примотать леску к грузу
        второй край на балконе петлёй обмотаться вокруг поручня, остальной метр подвести к SDR
        зачистить провода и припаять к проводам SMA (один край в SDR, второй зачистить - земля, сигнал)
        можно намытывать на катушку и спускать с балкона

    Альтернатива: 
        MiniWhip активная антенна (Aliexpress ~1500–2500₽)
    Дешевый и рабочий вариант:
        YouLoop (петлевая антенна, работает и внутри квартиры)

Для VHF/UHF (30–1000 МГц):
    Это FM-радио, авиация, такси, службы, метео, ISS.
    Диполь от RTL-SDR — годится, но выноси на подоконник или балкон.
    Антенна J-Pole, Discone или Ground Plane — под VHF/UHF, можно собрать самому.


=== Цифровой декодер =======================================================================
Ubuntu + SDR++
    SDR++ показывает сигнал и выводит звук
    Звук из SDR++ → в декодер (программа) через: 
        pavucontrol (PulseAudio) или
        qjackctl (Jack) - если потребуется точнее

Декодер обрабатывает аудиопоток и показывает данные
Можно декодировать данные, а не только слушать звук

    Назначение	        Частоты (МГц)	    Декодер / ПО	        Что показывает
    -------------------------------------------------------------------------------------------
    Пейджеры	        137–138 / 439–460	multimon-ng	POCSAG      текстовые сообщения
    ACARS (авиа)	    130.025–131.725	    acarsdec, multimon-ng	Тексты от самолётов (состояние, маршрут)
    AIS (корабли)	    161.975 / 162.025	rtl_ais, aisdecoder	    Позиции судов, скорость, маршрут
    NOAA APT	        137.1–137.9	        noaa-apt, WXtoImg	    Погода со спутников в виде карт
    DMR (цифра)	        438–450+	        DSD, DSD-FME, DSDcc	    Цифровая голосовая радиосвязь
    FT8, PSK31, RTTY	3–30 (HF)	        WSJT-X, Fldigi	        Радиолюбительские цифровые моды (тексты, QSO)
    ADS-B (авиа)	    1090	            dump1090	            Координаты и данные самолётов

ПО и установка:
1. multimon-ng (универсальный декодер)
    Поддерживает: POCSAG, ACARS, AFSK, AX.25, APRS
    sudo apt install multimon-ng
    использование:
        multimon-ng -a POCSAG1200 -t raw /dev/stdin

2. acarsdec
    sudo apt install acarsdec
    Запуск с RTL-SDR напрямую:
        acarsdec -v -p -r 0 131.725 131.525

3. rtl_ais
    sudo apt install rtl-ais
    rtl_ais -n

4. WSJT-X (для FT8, JT65 и др.)
    sudo apt install wsjtx
    Внутри настраиваешь аудиовход на "виртуальный микрофон", получаешь сигналы FT8 и прочие

5. Fldigi (RTTY, PSK31 и др.)
    sudo apt install fldigi
    Можно слушать радиолюбителей на КВ (HF)

6. noaa-apt + WXtoImg (для карт с метеоспутников)
    sudo apt install sox imagemagick
    pip install noaa-apt
    Затем ловишь APT-сигнал на частотах вроде 137.1, 137.62, 137.9125 МГц и декодируешь

Переделать звук:
    sudo apt install pavucontrol
    В SDR++ выбираешь вывод звука на виртуальный sink → декодер забирает этот звук

Настройки в SDR++
Модуляция:
    NFM — для большинства цифровых
    WFM (mono) — для широких вещательных
    Bandwidth: от 6 до 25 кГц
    Gain: ручной, лучше видеть «ровную» полосу без искажений
    AGC: выкл, лучше вручную
